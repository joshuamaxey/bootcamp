HTML Reference

Headings

Headings from 'h1' to 'h6' identify blocks and sections. Every page must have one (and ONLY one) 'h1.' This is important because web-crawlers search for the h1 on each page to label the search result. If they don't find one, you will have lower search rankings. If there is more than one, it can confuse the web crawler

Paragraphs

Paragraphs of text go in 'p' tags.

Lists

Unordered Lists

Unordered lists are marked with bullet oints and go in 'ul' tags. Each individual list item goes in a 'li' tag.

Types of Lists

  • Unordered lists ('ul')
  • Ordered lists ('ol')

Ordered Lists

Ordered lists are numbered and go in 'ol' tags. Each list item goes in a 'li' tag.

Anchors (links)

Anchor tags create links to other pages. The URL goes in the 'href' attribute. Search for more information on Google.

Setting 'target="_blank"' will cause our link to open in a new window. See this MDN page for more details.
